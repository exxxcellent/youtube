# YouTube copy

This is pet project is copy of YouTube

## Stack

Application:

-   React
-   Next.js

Styles:

-   Tailwind
-   SCSS
-   css-modules

Data fetching:

-   axios
-   SWR

Linting:

-   ESLint
-   Prettier

Control versions:

-   GitHub
-   Git

API:

-   YouTube v3

## To starts the project

Install dependencies:

```bash
npm install
```

Run dev mode:

```bash
npm run dev
```
